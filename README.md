Guess Game

This project is a simple number guessing game written in C#. In this game, the player must guess the number that the system (or the previous player) has entered. 
The player has 5 points, and each wrong guess reduces one point.
If the player guesses the number correctly before the points run out, they win. 
If all points are lost without a correct guess, the player loses.

Features:

Number guessing with limited points

Points decrease with each wrong guess

Win or lose detection at the end

Simple and easy to understand

Great for practicing conditions, loops, and input/output


How to Run:

1. Download or clone the source code.


2. Open the project in Visual Studio or any preferred C# environment.


3. Run the program and start guessing.


Technologies Used:

C#

Conditional statements

Loops

Console input and output


Project Purpose: To practice programming logic, point management, user input handling, and creating a basic number guessing game.
